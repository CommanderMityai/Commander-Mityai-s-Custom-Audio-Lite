 # Че это
Аддон для варкрафта с поддержкой проигрыша смешных и мемных звуков для сообщества "Казимир".

# Что конкретно он делает?
Проигрывает звуки в зависимости от внутриигрового события, будь то каст бладласта, смерть или прочее.

# Что это махина умеет?
То, что мне пришло в голову добавить. А именно:
- Конечно же, поддержка проигрыша звуков как главная механика.
- Три метода проигрыша звуков - прямой путь к файлу, кривой рандом (указать путь к папке, указать кол-во файлов внутри, обязательная нумерация файлов), выбор из библиотеки SharedMedia.
  
Конкретно реализованы звуки на следующие события:
- Получение бладласта и подобных ему баффов хасты.
- В зависимости от размера, вход в данж или вход в рейд.
- Боевой рес.
- Получение лута в зависимости от его редкости.
- Прерывание каста через кривой триггер-костыль.
- Смерть игрока.
- Фреймворк для звуков для любой ауры/баффа (и в теории дебаффа) в игре.
- Фреймворк для звуков для любого каста спелла в игре.

# Установка
Скачать зипник, прокинуть его содержимое в Interface\AddOns, все по стандарту.

# Переустановка при апдейтах
При нынешней архитектуре - чтобы всё было правильно прописано после апдейта, необходимо заходить в wow/_retail/wtf/вашакк/savedvariables и удалять настройки

Либо если сами уже что-то прописывали (кастомные ауры и баффы), прописывайте новое ручками, потому что в savedvariables сохраняется ваши новые спеллы.

Если что-то в дефолтах менялось - будет в чейнджлоге (Когда-нибудь сделаю локальный конфиг)

# Настройка
По вызову команды /cal (кал ха-ха) либо через параметры -> модификации. В окне увидите табы, в каждой можно выбирать включено ли событие.

В дефолтном конфиге всё должно быть настроено сразу "из-под капота".

Интуитивно понятный интерфейс, запутанно неочевидный выбор пути. 

Для одного файла - прописываете полностью без расширения, т.е Interface\AddOns\...\sounds\test\moizvuk, для рандома до папки, в которой должны лежать пронумерованные файлы (т.е 1.mp3 2.mp3 3.mp3 и т.д.) - Interface\AddOns\...\sounds\testrandom

Еще прикол - для рандома, чтобы запомнились настройки кол-ва файлов при ручном вводе, нужно ввести и нажать энтер, иначе не запомнит)

Есть кнопка теста для проверки правильности пути, для отладки можно раскомментить строчки 82-84.

Для кастомных аур/спеллов - ищете правильный айдишник, вставляет в поле, должно работать с ним.

# Известные проблемы
- Некоторый функционал еще не протестирован, и возможно имеются проблемы
- Есть ошибка от Ace3 если у вас включена близзардовская отладка, на работу не влияет
- Возможна нагрузка на пентиумы 3 - чек на ауры идет каждые 0.5 секунд
- БР не протестирован и возможно не работает если в команде нет друида, либо же любого персонажа, у которого есть заряды БРа.
- При внесении новоизменений и добавления новых звуков, нет локального сохранения всех настроек внутри аддона - они сохраняются в savedvariables внутри папки вашего аккаунта, и её необходимо удалять при обновлениях с предварительным отключением аддона.
